## Instalación
1. Clona el repositorio en tu máquina local.
2. Abre el projecto en tu IDE preferida.
3. Abre el terminal y ejecuta el siguiente comando:
   ```sh
   dotnet restore
   ```
4. Corre el proyecto:
   ```sh
   dotnet build
   ```
